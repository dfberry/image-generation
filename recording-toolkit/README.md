# Recording Toolkit

This package includes scripts, guides, and best practices for creating short demo videos using:
- OBS Studio
- Asciinema
- Screenity
- Peek

## Quick Start
- Use `scripts/record_cli.sh` or `scripts/record_cli.ps1` for fast CLI capture
- Use `scripts/convert_cast_to_gif.sh` or `scripts/convert_cast_to_gif.ps1` to convert recordings to GIF
- Use `docs/` for detailed instructions per tool
- See `docs/azure_demo_best_practices.md` for demo structure guidance
