## Architecture

Python CLI → OpenAI/Azure OpenAI → TSX component → npx remotion render → MP4

## Installation (User Action Required)

1. pip install -e .
2. cd remotion-project && npm install
3. Set OPENAI_API_KEY or Azure OpenAI env vars

## Usage

remotion-gen --prompt "A blue circle rotating 360 degrees" --output circle.mp4
remotion-gen --prompt "Text fading in" --quality high --duration 10 --output fade.mp4
